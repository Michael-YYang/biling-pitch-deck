Technical Architecture
AI 内容全链路四层技术架构
Layer 01
数据层
热点数据源
微博热搜、百度热搜、知乎热榜、B站热门，公开接口合规采集
合规爬取
用户历史数据
发布记录、互动数据（点赞/评论/收藏）、内容表现时序数据
MySQL / TiDB
平台内容数据
多平台账号绑定、内容日历、发布状态追踪、效果回流
S3 存储
向量知识库
高分案例 RAG 增强，病毒性内容特征向量化存储（Phase 2）
Phase 2
↓ 数据输入 ↓ 上下文注入 ↓ 历史参考 ↓ RAG 检索
Layer 02
AI 引擎层
LLM 文案引擎
Manus 内置 LLM（GPT-4o 级），多平台文案风格适配，结构化输出
invokeLLM
AI 视频引擎
火山引擎 doubao-seedance-1-0-pro / 1-5-pro，文生视频 + 图生视频（i2v）
已更新 · 国内版
AI 配图引擎
Manus 内置图像生成，多风格配图，与文案内容语义对齐
generateImage
Agent 编排引擎
多步骤 LLM 链式调用，热点→文案→配图→视频全自动流水线
tRPC Procedures
↓ 生成结果 ↓ 内容对象 ↓ 媒体资产 ↓ 任务状态
Layer 03
业务逻辑层
内容策略引擎
热点匹配、平台格式适配（9:16/16:9/1:1）、最优发布时间推荐
server/routers.ts
发布调度系统
内容日历管理、一键分发、剪贴板快捷、发布状态追踪
distributionRouter
数据分析引擎
互动率计算、内容表现对比、平台效果排行、趋势预测
analyticsRouter
用户权限系统
Manus OAuth 认证、角色权限（admin/user）、API 配额管理
protectedProcedure
↓ UI 数据 ↓ 操作响应 ↓ 状态更新 ↓ 权限校验
Layer 04
用户界面层
首页数据看板
渐变面积图、Sparkline 趋势卡片、实时数据概览
Home.tsx
内容生成工作台
AI 文案 + 配图 + Agent 全自动生成，多平台预览
Generate.tsx
AI 视频工作台
文生/图生视频、倒计时进度、批量多比例生成
VideoGenerator.tsx
发布 / 分析中心
内容日历、一键分发、效果追踪、互动数据录入
Publish / Analytics
技术栈：React 19 + TypeScript + tRPC 11 + Drizzle ORM + MySQL/TiDB + S3 存储 + Manus OAuth。 AI 能力全部通过内置 API 调用，无需用户配置密钥；视频引擎已切换至火山引擎国内版（ark.cn-beijing.volces.com），模型 ID 格式为 doubao-seedance-*。
前端：React 19 + Vite + TailwindCSS 4
后端：Express 4 + tRPC 11 + Superjson
数据库：MySQL/TiDB + Drizzle ORM
AI：Manus LLM + 火山引擎 Seedance + S3
